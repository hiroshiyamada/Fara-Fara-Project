文字認識API for Java サンプルプロジェクト
=========================================

- 本プロジェクトは Eclipse を利用してコンパイル・実行することができます。
  - Eclipse Downloads > http://www.eclipse.org/downloads/
- 本プロジェクトは 文字認識API for Java が依存するライブラリを /lib ディレクトリ以下に含んでいます。
- 本プロジェクトが依存するライブラリのライセンスについては、各Jarファイル名の末尾に ".license" が付与されているファイルを参照してください。
- 本プロジェクトは 文字認識API Webサイト ( http://recognize.jp/ ) のJava APIサンプルコードの具体例となっています。
